# Codex Build Prompt（AI会社ワークフロー）

このドキュメントは、ポートフォリオ構築をノンストップで進めるためのワークフローです。

## Phase 1: 要件整理
- 目的: ポートフォリオサイト作成
- 必須: 仮の作品を掲載
- 必須: X と GitHub アカウントへの導線
- 将来: AI関連成果物を継続掲載

## Phase 2: 実装
- 単一ページでヒーローセクション、作品一覧、掲載予定カテゴリを作成
- 仮作品を3件表示
- X / GitHub ボタンを設置

## Phase 3: 検証
- 静的ファイルに構文エラーがないことを確認
- ブラウザ表示を想定した最低限の動作確認を実施

## Phase 4: 引き渡し
- 変更内容の要約
- 実行コマンドと結果の記録
